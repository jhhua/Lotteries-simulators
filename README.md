#原理
非正式的赌博后台，只是模拟一下代码，体验买彩票的乐趣。肯定是有一些Bug的，通过bug可以预测结果，但没那个必要，因为运行速度很快，拿来利用不方便也没必要预测。不像买彩票会存入数据库，有机会就可以全部导出来。
